# 🎨 StudyQuest RPG — Prompts de Assets para Gemini Imagen
## Versão otimizada para Google Gemini / Imagen 3

---

## COMO USAR

1. Acesse **Google AI Studio** (aistudio.google.com) ou **Gemini Advanced**
2. Selecione o modelo **Imagen 3** quando disponível, ou use o Gemini 1.5 Pro com geração de imagem
3. Cole cada prompt abaixo **exatamente como está** — sem adicionar instruções extras
4. Para cada asset, gere **4 variações** e escolha a melhor
5. Exporte sempre em **PNG com fundo transparente** quando disponível

### Dicas gerais para o Gemini Imagen:
- Descreva o **estilo primeiro**, depois o conteúdo
- Use **inglês** — o Gemini Imagen performa melhor em inglês
- Seja **direto e técnico** — sem metáforas poéticas
- Se sair fundo branco em vez de transparente: peça novamente com "isolated on transparent background, no background"
- Se o resultado for genérico demais: adicione "high quality, detailed, professional" ao final

# REMOVER FUNDO — OPÇÕES GRATUITAS
# ══════════════════════════════════════════

Se o Gemini gerar com fundo branco ou cinza:

1. **remove.bg** (remove.bg) — 5 imagens grátis por mês, qualidade excelente
2. **Adobe Express** — remoção de fundo grátis com conta Adobe
3. **Canva** — ferramenta "Remover fundo" (plano gratuito tem limite)
4. **GIMP** (gratuito) — Filtros → Script-Fu → remover cor de fundo
